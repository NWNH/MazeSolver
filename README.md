# MazeSolver

基于 DFS 迷宫生成与 BFS、A* 寻路算法的可视化项目

---

## 一、项目背景

本项目是《数据结构与算法》课程大作业，旨在通过图形化界面展示经典图搜索算法的执行过程，并比较不同算法在迷宫寻路问题中的性能差异。

项目结合随机迷宫生成与路径搜索算法，使用户能够直观观察搜索过程，加深对图遍历、启发式搜索等知识点的理解。

---

## 二、项目功能

本项目实现了以下功能：

- 随机迷宫生成
- BFS（广度优先搜索）最短路径求解
- A* 启发式搜索最短路径求解
- Tkinter 图形界面可视化
- 搜索过程动画演示
- BFS 与 A* 搜索效率比较

---

## 三、核心算法

### 1. 随机迷宫生成（DFS）

迷宫采用递归深度优先搜索（Depth First Search）算法生成。

算法从起点开始，随机选择一个尚未访问的相邻节点，打通两点之间的墙壁，并递归访问该节点，直到所有节点均被访问。

#### 时间复杂度

```text
O(V)
```

其中 V 为迷宫中的节点数。

---

### 2. BFS 最短路径搜索

BFS 使用队列进行层次遍历。

由于迷宫可以看作无权图，因此 BFS 能保证找到从起点到终点的最短路径。

#### 使用的数据结构

- 队列（deque）
- 集合（set）
- 字典（dict）

#### 时间复杂度

```text
O(V+E)
```

---

### 3. A* 启发式搜索算法

A* 算法是一种带有启发函数的最优搜索算法。

评价函数定义为：

```text
f(n)=g(n)+h(n)
```

其中：

- g(n)：起点到当前节点的实际距离
- h(n)：当前节点到终点的曼哈顿距离

相比 BFS，A* 能够优先搜索更接近目标的位置，因此通常访问更少节点。

#### 使用的数据结构

- 优先队列（heapq）
- 字典（dict）
- 集合（set）

#### 时间复杂度

```text
O(E logV)
```

## 四、界面说明

项目使用 Tkinter 构建图形界面。

颜色说明：

|颜色|含义|
|---|---|
|黑色|墙壁|
|白色|可通行区域|
|绿色|起点|
|红色|终点|
|蓝色|搜索过程|
|黄色|最终路径|

用户可通过按钮实现：

- Generate：生成新的随机迷宫
- BFS：运行 BFS 搜索
- A*：运行 A* 搜索

```text
路径长度：41
```

### A*

